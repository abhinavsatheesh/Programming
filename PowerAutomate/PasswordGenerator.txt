Display.ShowMessageDialog.ShowMessage Title: $'''Password Generator''' Message: $'''Hi. Welcome to Password Generator. We can generate any sorts of Passwords for you. Press OK to Continue, and press Cancel to exit the program''' Icon: Display.Icon.Information Buttons: Display.Buttons.OKCancel DefaultButton: Display.DefaultButton.Button1 IsTopMost: False ButtonPressed=> ButtonPressed
IF ButtonPressed = $'''OK''' THEN
    Display.InputDialog Title: $'''Password Generator''' Message: $'''Enter the minimum length of the Password''' InputType: Display.InputType.SingleLine IsTopMost: False UserInput=> UserInput
    Display.ShowMessageDialog.ShowMessageWithTimeout Title: $'''Password Generator''' Message: $'''Generating your password, with Capital Letters, Small Letters, Numbers, and Special Characters. The length of the password will be %UserInput%.''' Icon: Display.Icon.Information Buttons: Display.Buttons.OK DefaultButton: Display.DefaultButton.Button1 IsTopMost: False Timeout: 3 ButtonPressed=> ButtonPressed2
    Text.Random UseUpperCaseLetters: True UseLowerCaseLetters: True UseDigits: True UseSymbols: True MinimumLength: UserInput MaximumLength: 10 RandomText=> RandomText
    Display.ShowMessageDialog.ShowMessage Title: $'''Password Generator''' Message: $'''Success. We generated a password for you. The generated password is
%RandomText%''' Icon: Display.Icon.Information Buttons: Display.Buttons.OK DefaultButton: Display.DefaultButton.Button1 IsTopMost: False ButtonPressed=> ButtonPressed3
END
